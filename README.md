# 🌐 OmniNews Pro 2026
> Portal de Notícias Full-Stack com Automação por IA.

## 🚀 Como Rodar o Projeto
1. **Backend:** No terminal, use `node servidor.js`
2. **Frontend:** Abra o `index.html` no navegador.
3. **IA Worker:** Execute `node ai-worker.js` para gerar notícias automáticas.

## 🛠️ Tecnologias
- HTML5/CSS3 (Interface Glassmorphism)
- Node.js (API Core)
- PostgreSQL (Base de Dados)
- Gemini AI (Automação de Conteúdo)
